Política de Privacidad

Esta aplicación utiliza la API Graph de Meta (Facebook e Instagram) para acceder únicamente a los datos estadísticos de una página de la que somos administradores o tenemos autorización.

La información recopilada se usa exclusivamente con fines analíticos y de reporte interno. No se recopilan, almacenan ni comparten datos personales de usuarios.

No compartimos esta información con terceros ni hacemos uso comercial de los datos.

Para cualquier duda, puedes contactarnos en: analistadedatos@carvajalips.com
